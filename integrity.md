## 🔍 How This Was Built

This protocol stands on the shoulders of giants. Key components were forked from:

- **Privacy Pools** (Ameen Soleimani): ZK circuits for compliant anonymity  
- **Semaphore** (Ethereum Foundation): Merkle tree logic  
- **Uniswap v4**: Hook architecture for shielded swaps  

I’ve modified, tested, and connected these systems into a new privacy-first economy.  
The code is battle-tested (see [SECURITY.md]), but contributions and audits are welcome!
